elxElastixVersion

Return the Elastix version

Contents

  • Syntax
  • Input arguments
  • Output arguments
  • See also
  • License

Syntax

Version = elxElastixVersion(elxConf)

Input arguments

  • elxConf (StrToolboxConf Structure): Elastix configuration

Output arguments

  • Version (1x2 vector): The major and minor version of Elastix. [0 0] if not found.

See also

elxElastix, elxTransformix, StrToolboxConf
