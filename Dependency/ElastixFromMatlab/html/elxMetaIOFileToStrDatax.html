elxMetaIOFileToStrDatax

Read a MetaIO file and return the image in a StrDatax structure with additional fields.

Contents

  • Syntax
  • Input arguments
  • Output arguments
  • Description
  • See also
  • License

Syntax

[StrDatax, Success, Message] = elxMetaIOFileToStrDatax(Filename, PermuteAxes)

Input arguments

  • Filename (string): The MetaIO filename
  • PermuteAxes (boolean): optional argument. If true permute the first two axes. Its default value is false.

Output arguments

  • StrDatax (StrDatax): The image.
  • Success (logical): true on success.
  • Message (string): Error message

Description

Read a MetaIO file and return the image in a StrDatax structure with additional fields.

See also

elxElastix, elxTransformix, elxStrDataxToMetaIOFile
